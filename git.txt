cls - очистка консоли
git --version версия гита
git --help справочник
dir что есть в директории
git init инициализировать репозиторий (создалась системная папка гит) ее по умолчанию не видно
git status что происходит в папке. ветка, не было коммитов
git add <название файла> указываем гиту следить за этим файлом и еще раз git status
git rm --cached <file>... отменить отслеживание файла
что бы добавить все что есть в папке (а не по одному или конкретный) - git add . (с точкой)
git commit -m "текст......."
.gitignore не забывать. сюда же можно добавлять исключаемые папки внутри директории
git branch <название> - новая ветка
git checkout <название> переключится на другую ветку
git config --global user.name 
git config --global user.email
git remote add origin https://github.com/EgoDaemon/garbage.git указываем куда заливаем - кажется только один раз, но это не точно
git push -u origin master (или другое название ветки)
git clone https://github.com/EgoDaemon/garbage.git - клонировать себе на комп репу (не забыть перейти предварительно в нужную директорию на компе)


git pull команда для проверки серверных изменений разрабатываемого файла
